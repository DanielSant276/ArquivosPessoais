O ng-controller é responsável por anexar uma classe a view da aplicação. Os módulos são utilizados para separar as partes do código, criando uma organização para a estrutura do código. Os módulos são semelhantes a pacotes/bibliotecas. No exemplo dados temos 2 controles: um chamado mainApp.js para inicializar o controller da aplicação e um modulo chamado questao4.js que é usado para definir o controller.
